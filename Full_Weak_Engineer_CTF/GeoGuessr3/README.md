## Description(問題文)

この写真を撮っている人の座標を指定してください。
また、座標を直接指定すると誤差の許容範囲が表示されません。マウスクリックでご確認ください。

Please specify the coordinates of the person who took this photo.
Note that if you enter the coordinates directly, the tolerance zone will not be shown. Please check it by clicking with the mouse.

## Solution
From the “$10” sign on a shop called Phonix, I guessed the photo was taken in the US.
Also, since it had a Chinatown-like vibe, I suspected it might be in Chinatown.

Using Google Lens here reveals that there’s a place in Los Angeles Chinatown with a very similar atmosphere.
If you search for “Los Angeles phoenix clothing store” and look around, you can find this location:
