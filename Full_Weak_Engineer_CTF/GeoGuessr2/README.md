## Description(問題文)

この写真を撮っている人の座標を指定してください。
また、座標を直接指定すると誤差の許容範囲が表示されません。マウスクリックでご確認ください。

Please specify the coordinates of the person who took this photo.
Note that if you enter the coordinates directly, the tolerance zone will not be shown. Please check it by clicking with the mouse.

## Solution
It looked like the language might be German.

When you search for “kaisertor frankfurt”, you’ll find the following Instagram account:
https://www.instagram.com/kaisertor_frankfurt/?hl=en

Since the address is listed there, you can match the location by comparing Google Maps with the photo.
